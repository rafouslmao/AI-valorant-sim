# Valorant eSports Simulator (MVP)

Vanilla JS + Vite simulator with split core logic/storage/ui inspired by zengm-legacy architecture.

## Team Data

The default team list is preloaded with Tier 1 / VCT-style organizations (Americas, EMEA, Pacific, China) to match the request for a competitive top-tier ecosystem.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.
